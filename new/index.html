Regular Expressions (Regex)

The "Alien Language" of pattern matching. Essential for Forms & Search.

1. The Basics

  • /abc/ : Matches "abc" anywhere
  • ^abc : Must START with abc
  • abc$ : Must END with abc
  • . : Any single character

2. Special Codes

  • \d : Any Digit (0-9)
  • \w : Any Word Char (A-Z, a-z, 0-9, _)
  • [A-Z] : Any Uppercase Letter
  • {8,} : Minimum 8 chars
